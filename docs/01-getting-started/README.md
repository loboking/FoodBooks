# Getting Started

FoodBooks 프로젝트 시작 가이드입니다.

## 필수 요구사항

### 소프트웨어
- **웹 브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)
- **Node.js**: 14+ (Firebase CLI 사용 시)
- **Git**: 버전 관리용

### 서비스 계정
- **Firebase 계정** (커뮤니티 기능 사용 시)
- **Pexels API 키** (이미지 검색 사용 시)

## 설치

### 1. 저장소 클론

```bash
git clone <repository-url>
cd FoodBooks
```

### 2. Firebase CLI 설치 (선택사항)

커뮤니티 기능 및 배포를 사용하려면 Firebase CLI가 필요합니다.

```bash
npm install -g firebase-tools
```

### 3. Firebase 로그인

```bash
firebase login
```

### 4. Firebase 프로젝트 연결

기존 프로젝트를 사용하거나 새 프로젝트를 생성합니다.

```bash
# 기존 프로젝트 확인
firebase projects:list

# 이미 .firebaserc에 설정되어 있음 (foodbooks-community)
```

## 로컬 실행

### 방법 1: Firebase CLI (권장)

```bash
npm run serve
# 또는
firebase serve
```

기본 포트: `http://localhost:5000`

### 방법 2: Python HTTP 서버

```bash
python -m http.server 8000
# 또는 Python 3
python3 -m http.server 8000
```

기본 포트: `http://localhost:8000`

### 방법 3: Node.js HTTP 서버

```bash
npx serve
# 또는
npm install -g serve
serve
```

기본 포트: `http://localhost:3000`

## 빠른 시작

### 1. 애플리케이션 접속

브라우저에서 `http://localhost:5000` (또는 해당 포트) 접속

### 2. 샘플 데이터 로드

처음 실행 시 샘플 레시피 데이터를 로드할 수 있습니다:

1. **설정** 탭 선택
2. **샘플 데이터 로드** 버튼 클릭
3. 35개 레시피가 자동으로 추가됩니다

### 3. 레시피 탐색

- **홈** 탭: 카테고리별 레시피 보기
- **레시피** 탭: 전체 레시피 검색 및 필터
- 레시피 카드 클릭 시 상세 정보 확인

### 4. 레시피 추가

1. **레시피** 탭에서 **+** 버튼 클릭
2. 레시피 정보 입력:
   - 제목
   - 카테고리 & 서브카테고리
   - 난이도 (쉬움/보통/어려움)
   - 준비 시간 & 조리 시간
   - 인원
   - 재료 (이름, 양, 단위)
   - 조리 단계
   - 태그
3. 이미지 검색 (선택사항)
4. **저장** 버튼 클릭

### 5. 이미지 검색

레시피 작성 중:

1. **이미지 검색** 버튼 클릭
2. 음식명 입력 (한글 또는 영어)
3. 검색 결과에서 이미지 선택
4. 선택한 이미지가 자동으로 레시피에 추가됨

### 6. 커뮤니티 사용

1. 브라우저에서 `community.html` 접속
2. 닉네임 설정
3. 게시물 작성 및 공유
4. 다른 사용자의 레시피 확인
5. 댓글 작성

## 주요 기능 테스트

### 검색 & 필터

1. **레시피** 탭에서 검색창 클릭
2. 텍스트 검색: 레시피 제목, 설명, 재료로 검색
3. 필터 적용:
   - 카테고리: 한식, 중식, 일식, 양식 등
   - 태그: #간편요리, #다이어트 등
   - 난이도: 쉬움/보통/어려움
   - 조리 시간: 30분 이내, 1시간 이내 등
4. 정렬: 최신순, 이름순, 난이도순, 시간순

### 요리 모드

1. 레시피 상세 페이지에서 **요리 시작** 버튼 클릭
2. 전체 화면으로 단계별 가이드 표시
3. 각 단계 완료 시 체크
4. **완료** 또는 **나가기** 버튼으로 종료

### 쇼핑리스트

1. 레시피 상세 페이지에서 **장보기 추가** 버튼 클릭
2. 인원 조정 (자동으로 재료량 계산)
3. **쇼핑** 탭에서 통합 리스트 확인
4. 재료 체크 또는 삭제

### 데이터 백업 & 복원

**백업**:
1. **설정** 탭 선택
2. **데이터 내보내기** 버튼 클릭
3. JSON 파일 다운로드

**복원**:
1. **설정** 탭 선택
2. **데이터 가져오기** 버튼 클릭
3. JSON 파일 선택
4. 확인

## 문제 해결

### IndexedDB 오류

**증상**: 레시피가 저장되지 않음

**해결**:
1. 브라우저 개발자 도구 열기 (F12)
2. Application > Storage > IndexedDB 확인
3. 프라이빗 모드가 아닌지 확인
4. 브라우저 재시작

### Firebase 연결 오류

**증상**: 커뮤니티 게시물이 로드되지 않음

**해결**:
1. `.firebaserc` 파일 확인
2. Firebase 프로젝트가 활성화되어 있는지 확인
3. 네트워크 연결 확인
4. Firebase Console에서 Realtime Database 규칙 확인

### 이미지 검색 오류

**증상**: 이미지 검색 결과가 없음

**해결**:
1. Pexels API 키 확인 (js/image-verification.js)
2. CORS 오류 확인 (개발자 도구 Console)
3. 네트워크 연결 확인
4. API 할당량 확인
